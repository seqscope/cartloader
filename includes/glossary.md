*[ST]: Spatial transcriptomics
*[SGE]: Spatial Digital Gene Expression
*[µm]: micrometer
*[UMI]: Unique Molecular Identifier
*[MEX]: Matrix Exchange (10x sparse matrix format)
*[DNB]: DNA Nanoball (DNB)
*[MERFISH]: Multiplexed Error Robust Fluorescence In Situ Hybridization
*[FICTURE]: Segmentation-free method inferring latent spatial factors from SGE (Si et al., Nature Methods 2024)
*[PMTiles]: Single-file archive format for map tiles and web visualization
*[UMAP]: Uniform Manifold Approximation and Projection; nonlinear dimensionality reduction
*[PCA]: Principal Component Analysis; linear dimensionality reduction
*[LDA]: Latent Dirichlet Allocation
*[parquet]: Columnar storage format for large tabular datasets
*[DE]: Differential Expression; comparing groups to find significantly changed genes
*[OME‑TIFF]: Microscopy TIFF format from OME with rich metadata and pyramids
*[DAPI]: Fluorescent DNA stain that labels cell nuclei
*[MIP]: Maximum intensity projection of a z‑stack
*[GeoTIFF]: TIFF image with embedded geospatial referencing
*[GeoJSON]: JSON-based format for vector geospatial features
*[georeference]: Assign image pixels to spatial coordinates or a reference system
